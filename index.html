Kemampuan Akhir yang Diharapkan : Mahasiswa mampu mencari peluang bisnis sesuai dengan permintaan dan penawaran di sekitar kampus.

Langkah-langkah Pengerjaan Tugas

  • Mengumpulkan lebih dari 10 gagasan bisnis melalui survey kebutuhan dan keinginan di sekitar kampus.
  • Menyempurnakan ide dengan observasi kepada keluarga atau teman yang menjalankan bisnis relevan.
  • Memilih 10 gagasan yang paling mungkin (possible) dijalankan.
  • Menyesuaikan ide bisnis dengan keahlian, latar belakang pendidikan, pengalaman, atau hobi.
  • Memilih 5 gagasan paling murah (low cost) berdasarkan kemampuan modal.
  • Memilih 1 gagasan terbaik (the best) dari aspek organisasi, pemasaran, produksi, dan keuangan.

Observasi ke Keluarga atau Teman yang Sudah Menjalankan Bisnis

  • Keluarga menjalankan bisnis laundry di dekat kampus, permintaan tinggi karena mahasiswa sibuk.
  • Teman menjalankan bisnis mie ayam take away, selalu ramai pesanan terutama mahasiswa yang tidak sempat masak.
  • Teman menjadi reseller pakaian online, modal kecil dan pemasaran lewat media sosial.
  • Teman membuka jasa print online, banyak mahasiswa butuh print tugas tanpa harus ke luar kos.
  • Keluarga menjalankan toko alat tulis, laris karena kebutuhan mahasiswa setiap hari.

5 Ide Bisnis Paling Murah (Low Cost)

Modal Patungan dengan Teman

Modal patungan bersama teman merupakan salah satu strategi yang efektif untuk memulai usaha, terutama bagi mahasiswa yang memiliki keterbatasan modal sendiri. Dengan modal bersama, beban finansial dapat dibagi sehingga risiko kerugian menjadi lebih kecil dan peluang untuk mengembangkan bisnis menjadi lebih besar.

Modal patungan juga memungkinkan pembagian tugas sesuai keahlian masing-masing, sehingga operasional bisnis bisa berjalan lebih efisien dan terorganisir.

  1. Jasa Desain Grafis : Hanya membutuhkan laptop dan software desain (bisa pakai versi gratis).
  2. Jasa Install Ulang Windows : Modal utama adalah keahlian dan flashdisk, tidak perlu tempat usaha.
  3. Bisnis Pulsa dan Paket Data : Modal awal sangat kecil, bisa dijalankan dari mana saja.
  4. Jasa Deep Cleaning Kamar Kost : Hanya bermodalkan sapu, kain pel, dan pembersih saja.
  5. Bisnis Hampers dan Bucket : Hanya bermodalkan dakat merias, merangkai, dan menata bunga-bunga.

Gagasan Terbaik: Jasa Install Ulang Windows

Alasan Pemilihan:
  • Modal utama adalah keahlian dan flashdisk, tidak perlu tempat usaha.
  • Cukup sekali beli flashdisk dapat digunakan berkali-kali untuk install ulang.
  • Masyarakat membutuhkan Install Ulang Windows untuk memperbarui ke Windows yang terbaru.
  • Mahasiswa melakukan Install Ulang Windows sebab laptopnya sudah terlalu lambat dikarenakan sering mendownload game dari situs ilegal.
  • Alasan lainnya adalah Windows merupakan sistem operasi yang paling sering di serang virus, perlunya Install Ulang agar dapat mengembalikan kembali performa laptop.
  • Risiko rendah, pasar luas, dan 80% laptop di masyarakat dan mahasiswa menggunakan laptop Windows.
